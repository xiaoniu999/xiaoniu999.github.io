顺序容器有哪些

顺序容器

C++ Premier 5th 书中定义了6种顺序容器，它们分别是：

  • vector
  • deque
  • list
  • forward_list
  • array
  • string

但stack也是顺序容器，这篇文章主要讲讲stack。
补充：stack称为容器适配器，是基于上述容器实现的，标准库中是基于deque实现。

stack容器的操作有哪些

  • empty() 判断栈是否为空
  • size() 返回栈中元素的数量
  • top() 返回栈顶元素
  • push(elem) 向栈顶添加元素
  • pop() 移除栈顶元素

stack的实现

stack的实现是利用了vector，vector的实现是利用了数组，所以stack的实现是利用了数组。
